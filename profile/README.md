# IGG Korea

![IGG Korea][hero-image]

## 한국 인디게임 개발자에게 필요한 연결을 만듭니다

[![IGG-K Discord 참여][discord-badge]][discord]
[![support@iggk.net으로 문의][contact-badge]][email]

> **IGG Korea (IGG-K)**는 Google Play의 후원을 받는 글로벌 Indie Games
> Group의 한국 챕터입니다. 인디게임 개발자가 동료와 만나고, 작품을 알리고,
> 산업과 글로벌 기회에 연결될 수 있도록 커뮤니티와 행사를 운영합니다.

## 우리가 만드는 연결

| 영역 | 만드는 연결 |
| :--- | :--- |
| 🤝 개발자 | 동료 개발자와 교류하고 협업합니다. |
| 🎮 게임 | 이용자와 업계에 작품을 소개합니다. |
| 💡 실무 | 개발·기획·사업·기술 경험을 나눕니다. |
| 🌏 생태계 | 플랫폼·퍼블리셔·기관·글로벌 기회와 연결합니다. |

## 커뮤니티와 행사가 순환합니다

**Discord 커뮤니티**에서 관계와 필요를 발견하고 →
**네트워킹·세미나·개발 챌린지·전시**에서 실제 만남과 기회를 만들고 →
그 결과를 다시 커뮤니티의 지식과 연결로 축적합니다.

## 주요 프로그램

- **IGG with Seoul Game Alley** — 개발자·기관·업계가 만나는 네트워킹과 쇼케이스
- **Build with AI for Games** — 개발과 비개발 영역의 실무 지식 공유
- **서울 인디게임 개발 챌린지** — 집중 제작과 후속 기회를 잇는 프로그램
- **ALT+G** — 개발자, 게임, 이용자, 문화가 만나는 인디게임 페스티벌

## 함께하세요

게임을 개발 중이거나 개발을 준비하고 있다면 누구나 IGG-K Discord에
자유롭게 참여할 수 있습니다. 개인 개발자·팀·회사 소속 개발자뿐 아니라
학생과 예비 개발자도 환영합니다.

[![IGG-K Discord 참여하기][join-badge]][discord]

> Discord 참여는 커뮤니티 활동 참여를 뜻하며, 법적 회원 자격이나 IGG-K의 공식 인증을 의미하지 않습니다.
